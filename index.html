GLOBAL REACH FOR GAMING COMMUNITIES

Based in Sydney, Elo Bot is built for Discord servers around the world. Whether you’re hosting tournaments or casual PvP events, Elo Bot helps you manage rankings, track performance, and keep your server running smoothly.


With automated tools and easy setup, server admins can save time and give players a better experience — all without writing any code.

OUR MISSION

Simple. Competitive. Discord-ready.


Elo Bot was made to help gaming communities run PvP events more easily. Our goal is to offer powerful tools that are easy to set up, customizable, and fun to use.

- Real-time ELO Rankings
- Server Moderation Tools
- Role Automation

We aim to support both casual and competitive communities by providing features that bring structure and engagement to Discord.

FEATURES

ELO Ranking System

Tracks player performance using an ELO system for fair and competitive matchups.

Live Leaderboard

Automatically updates ranks after each match and displays them in a clean leaderboard.

Built-in Moderation

Keep your server safe with built-in commands like mute, kick, ban, and warnings.

Match History Tracking

Stores all match results so you can review or undo them anytime.

Auto Role Assignment

Players can automatically earn roles when they reach certain ELO levels.

Easy Setup

Use slash commands to set up your server in minutes — no coding needed.
