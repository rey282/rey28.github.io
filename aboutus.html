Who We Are

Elo Bot is a project built by two developers from Sydney, Australia, designed to help Discord gaming communities manage PvP events with ease.

We created Elo Bot to give small and medium Discord servers access to professional features like live ELO tracking, leaderboards, and server moderation — without needing coding experience.

Learn More

What We Do

ELO Ranking System: We help servers track player performance and competition with our accurate and fair ELO algorithm.

Live Leaderboard: We make it easy to display rankings that update automatically after each match.

Moderation + Automation: Our bot includes moderation tools and role-based automation to help server admins save time.

Match History & Reporting: Full history of past matches, easy to access and roll back if needed.

Global Reach

Although we’re based in Sydney, Elo Bot is made for communities all around the world. As long as you use Discord, you can use Elo Bot.

Our Vision

To make competitive gaming more fun, fair, and accessible to every Discord community, no matter how big or small.

Our Mission

Powerful tools, simple setup. We want to help communities grow stronger with better ranking systems, easier moderation, and clear competition. Elo Bot aims to save server admins time while giving players a better experience.
